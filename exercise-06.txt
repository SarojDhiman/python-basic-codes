Q2 : Write a Program to display the Fibonacci sequence up to n-th term where n is provided by
#the user
def recur_factorial(x):
    if x <= 1:
        return x
    else:
        return(x*recur_factorial(x-1))

x = int(input("value"))

print(recur_factorial(x))

Q6: Write a Python program to find the sum of natural numbers up to n using recursive function

    if x <= 1:
        return x
    else:
        return(x + recur_sum(x-1))
